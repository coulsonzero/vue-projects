# Vue 3 + Vite

## QuickStart

```sh
npm install
# if you need `vue-router`:
# yarn add vue-router@next -S

# if you need `vuex`:
# yarn add vuex@next --save

# if you need `scss`:
# yarn add -D sass

yarn dev
```


## Create a new vue3 vite project

```
yarn create vite <project-name> --template vue
```
